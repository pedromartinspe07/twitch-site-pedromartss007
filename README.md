# Site do Canal Twitch

Site pessoal para canal da Twitch com tema Frutiger Aero combinado com minimalismo e surrealismo.

## Características

- Design inspirado no estilo Frutiger Aero
- Elementos minimalistas e surrealistas
- Totalmente responsivo
- Animações suaves e efeitos visuais
- Inclui página de regras com PDF
- Desenvolvido com React

## Estrutura do Site

1. **Cabeçalho**: Logo, navegação e indicador de live
2. **Seção Hero**: Apresentação do canal e estatísticas
3. **Sobre**: Informações sobre o streamer e conteúdo
4. **Programação**: Agenda semanal de streams
5. **Links**: Redes sociais e links importantes
6. **Regras**: Regras do canal com opção de download em PDF
7. **Rodapé**: Informações de contato e links adicionais

## Como Executar

### Pré-requisitos
- Node.js (versão 14 ou superior)
- npm ou yarn

### Instalação

1. Extraia o arquivo ZIP
2. Abra o terminal na pasta do projeto
3. Execute: `npm install` ou `yarn install`
4. Execute: `npm start` ou `yarn start`

O site será aberto em `http://localhost:3000`

### Para Produção

Execute `npm run build` ou `yarn build` para criar a versão de produção.

## Personalização

1. **Texto**: Edite os textos nos componentes React
2. **Cores**: Modifique as cores nos arquivos CSS
3. **Links**: Atualize os links nas propriedades dos componentes
4. **PDF**: Coloque o arquivo `rules.pdf` na pasta `public/`
5. **Imagens**: Substitua os ícones por imagens reais

## Tecnologias Utilizadas

- React 18
- CSS3 (com animações e gradientes)
- Font Awesome (ícones)
- Google Fonts (Inter)

## Observações

- O PDF das regras deve ser colocado na pasta `public/` como `rules.pdf`
- Os links para redes sociais são placeholders - atualize com seus links reais
- As cores e estilos podem ser facilmente modificadas nos arquivos CSS
